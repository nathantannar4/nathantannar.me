## Tech Stack

- Vapor 3
- Swift 4.1

> Preview it live at [https://nathantannar.me](https://nathantannar.me)

## Build

```
vapor build
```

## Run

```
vapor run // Serves over localhost:8000
```

## Deployment

The Vapor build is hosted on a Google Cloud Ubuntu instance proxied behind NGINX webserver
